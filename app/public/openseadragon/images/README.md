This folder is the default location for OpenSeadragon control images.

If you want to use the default button icons locally, copy the contents of
`node_modules/openseadragon/build/openseadragon/images` into this folder.

Alternatively set `prefixUrl` in the viewer to point to a CDN or another path.

Example:
  - Copy `zoom-in.svg`, `zoom-out.svg`, `home.svg`, etc. into this folder.
